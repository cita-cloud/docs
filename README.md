# `CITA-Cloud`文档

## 使用说明

本文档使用[MkDocs](https://www.mkdocs.org/)开发。

### 安装

```bash
pip install mkdocs
```

### 预览

启动本地预览服务器。

```bash
mkdocs serve -a 0.0.0.0:8000
```

访问`http://127.0.0.1:8000/`就可以看到文档的预览效果。
